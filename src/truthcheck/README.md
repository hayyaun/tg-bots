## Setup

### 1. Create a Telegram Bot

1. Talk to [@BotFather](https://t.me/BotFather) on Telegram
2. Create a new bot with `/newbot`
3. Get your bot token
4. **Important**: Enable inline mode:
   - Send `/setinline` to BotFather
   - Select your bot
   - Set a placeholder text like "Type statement to fact-check..."
   - Send `/setinlinefeedback` to enable inline feedback

### 2. Get OpenAI API Key

1. Go to [OpenAI Platform](https://platform.openai.com/)
2. Create an account or sign in
3. Go to API Keys section
4. Create a new API key
5. Copy the key (you won't be able to see it again)

### 3. Configure Environment

Add to your `.env` file:

```bash
TRUTHCHECK_BOT_KEY=your_bot_token_here
OPENAI_API_KEY=your_openai_api_key_here
```

### 4. Run the Bot

```bash
npm run dev
```

## Usage

### Method 1: Inline Mode (Any Chat)

In any Telegram chat (private or group):

1. Type `@your_bot_username` followed by the statement you want to fact-check
2. The bot will show you the fact-check result
3. Tap on the result to send it to the chat

**Example:**
```
@truthcheck_bot The Earth is flat
@truthcheck_bot Water boils at 100 degrees Celsius at sea level
```

### Method 2: Reply to Messages (Private Chat)

1. Start a private chat with the bot
2. Reply to any message you want to fact-check
3. The bot will analyze the replied message and provide a fact-check report

### Method 3: Forward Messages (Private Chat)

1. Start a private chat with the bot
2. Forward any message you want to fact-check
3. The bot will analyze the forwarded message and provide a fact-check report

### Commands

- `/start` - Welcome message and instructions
- `/help` - Show help and usage information

## How It Works

The bot uses ChatGPT to analyze statements and provides:

1. **Verdict**: Clear classification (TRUE, MOSTLY TRUE, PARTIALLY TRUE, MOSTLY FALSE, FALSE)
2. **Explanation**: Detailed reasoning behind the verdict
3. **Evidence**: References to reliable sources and established facts
4. **Correction**: If false, explains what the correct information is

### Fact-Checking Criteria

The bot references reliable sources such as:
- Peer-reviewed scientific journals
- Official government records and statistics
- Established academic institutions
- Recognized expert consensus
- Verified historical records
- Reputable news organizations with fact-checking standards

## Limitations

- Fact-checking is based on available information and may not cover all contexts or recent developments
- The bot focuses on factual claims, not opinions
- Results are generated by AI and should be used as a starting point for verification
- Some statements may require additional research or expert consultation

## Technical Details

- **Model**: GPT-3.5-turbo
- **Temperature**: 0.3 (for consistent, factual responses)
- **Max Tokens**: 800 (for detailed but concise responses)
- **Response Format**: HTML-formatted messages with clear structure
